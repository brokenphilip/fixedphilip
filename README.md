# fixedphilip
fixedphilip is a general-purpose Discord bot written in C++, utilizing the [D++](https://dpp.dev/) library.

> [!NOTE]
> This repository is currently an **early work-in-progress**, any and all information is subject to change without notice and may be missing or incorrect.

> [!IMPORTANT]
> In its current state, the bot is designed for **personal use only** - feature requests will likely not be considered. The repository remains public solely for my own convenience in deployment and development.

> [!WARNING]
> In its current state, there are many unresolved issues regarding the building process of this repository.
> 
> In addition, the building process is largely undocumented, and the target machine is a very specific niche Armbian device - no tests are being or will be made on other architectures.

> [!CAUTION]
> In its current state, the quality of this repository is **very poor**, and it basically runs on magic - please avoid using it, even as a learning reference, and *especially* in a production environment.
